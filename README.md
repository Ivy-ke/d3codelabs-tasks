DecodeLabs Industrial Training Kit — Security Projects

This repository contains Python security projects I developed during my internship as part of the DecodeLabs Cybersecurity Analyst Training Program. Each project builds on the skills learned in the previous one, starting with basic programming logic, progressing to cryptography, and finally applying those concepts to phishing detection and threat analysis.

The projects were completed as part of my learning journey to strengthen my practical programming and cybersecurity skills.

Projects
Project 1: Password Strength Checker

A command-line application that evaluates the strength of a password based on its length, character variety, estimated entropy, and common weak password patterns. The tool classifies passwords as Weak, Medium, or Strong and provides a detailed strength report.

Skills Demonstrated:

Python string handling
Conditional logic
Password entropy concepts
Input validation
Basic defensive programming
Project 2: Caesar Cipher

A command-line application that encrypts and decrypts text using the Caesar Cipher. This project introduces the basic principles of symmetric encryption, modular arithmetic, and character manipulation using ASCII values.

Skills Demonstrated:

Encryption and decryption concepts
ASCII manipulation
Modular arithmetic
Algorithm design
Python programming

Project 3: PhishGuard

A command-line phishing awareness tool that analyzes emails and messages for common phishing indicators, including suspicious keywords, misleading URLs, and sender/domain mismatches. The tool generates a risk score and classifies messages as Safe, Suspicious, or Malicious, while explaining the reasons behind the classification.

Skills Demonstrated:

Threat analysis
Pattern recognition
Regular expressions
Basic phishing detection
Security awareness concepts
