# DecodeLabs  — Security Projects

Python security tools built during my internship as part of the DecodeLabs Cybersecurity Analyst training track. Each project builds on the last — starting from foundational logic, moving into cryptography, then into threat analysis.

## Projects

### [Project 1: Password Strength Checker](./project-1-password-checker)
Evaluates password strength (Weak/Medium/Strong) based on length, character variety, and estimated entropy. Flags common weak/leaked password patterns.

**Skills:** String handling, conditional logic, entropy calculation

---

### [Project 2: Caesar Cipher](./project-2-caesar-cipher)
Encrypts and decrypts text using a classic substitution cipher, demonstrating symmetric encryption fundamentals and modular arithmetic.

**Skills:** Encryption concepts, ASCII manipulation, algorithmic logic

---

### [Project 3: PhishGuard](./project-3-phishguard)
Analyzes emails/messages for phishing indicators — suspicious keywords, lookalike URLs, sender/domain mismatches — and produces a risk score with a Safe/Suspicious/Malicious classification.

**Skills:** Threat analysis, regular expressions, pattern recognition
